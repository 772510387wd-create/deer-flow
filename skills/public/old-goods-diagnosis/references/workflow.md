# 老品诊断与建议 — 流程参考

本文件为 SKILL.md 中 Workflow 的补充说明，供需要时按需加载。

## 依赖关系小结

- 知识库查询 **依赖** 新老品标识查询结果。
- 生成诊断结果 **依赖** 目标信息、业务表现数据、知识库、OKR 信息全部就绪。
- 写入诊断结果 **依赖** 生成诊断结果、目标信息、业务表现数据。

## 与后端对接说明

若通过 API 或脚本对接 decision_support_agent 或其它后端，各步骤对应工具名称为：

1. 商品新老品标识查询
2. 老品诊断与建议-查询目标信息
3. 老品诊断与建议-查询业务表现数据
4. 基于新老品标签查询知识库
5. 老品诊断与建议-查询OKR信息
6. 老品诊断与建议-生成诊断结果
7. 老品诊断与建议-写入诊断结果

详细输入输出契约以各工具/接口定义为准。
